# Веб-версия настольной игры "Нуар"

Используя это приложение Вы можете сыграть в настольную игру "Нуар" в созданной Вами комнате.

### Как запустить:

```
pip install -r requirements.txt
docker run -p 6379:6379 -d redis:5
python manage.py runserver
```

Перейти по адресу: <em>http://127.0.0.1:8000/ </em>
